Introduction

python快速应用教程

本教程旨在为读者提供：

  • python简明入门教程
  • python常用库与命令手册
  • python常用应用入门教程

本教程适合以下读者：

  • 无任何编程基础，想要以python作为入门语言；
  • 有其他语言的编程基础，想要转向python进行特定应用的开发；
  • 有python的编程基础，想要进行特定应用的开发。

本教程不适合以下读者：

  • 想要探究程序设计的深层原理；
  • 想要探究python语言的高级特性及原理；
